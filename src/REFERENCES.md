## API Coverage

- Workbook: create, open, save, get info
- Sheets: list, add, remove, rename, duplicate, get info
- Read: get cell, get range, get row, get column, get all data
- Write: set cell, set range, insert row, insert column, delete row, delete column
- Format: format cell, format range, set column width, set row height, merge cells, unmerge cells
- Formulas: set formula, recalculate
- Tables: create table, add table row
- Charts: create chart (bar, line, pie, scatter)
- Export: CSV, JSON, PDF
- Filters: add autofilter, sort column
